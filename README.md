# PhoenixWP Gift Product

WooCommerce extension: automatically add a **free gift product** to the cart.

## Requirements

- WordPress 6.7+ (tested up to 7.0), PHP 8.2+
- WooCommerce 8.0+ (tested up to 10.8.1)
- PhoenixWP Core (recommended)

## Free tier (MVP)

- Enable / disable
- One gift product (simple)
- Optional minimum cart subtotal

## Setup

```powershell
cd phoenix-gift-for-woocommerce
C:\php\php.exe ..\phoenix-wp-core\composer.phar install
```

Copy to `wp-content/plugins/phoenix-gift-for-woocommerce`, activate Core + Gift + WooCommerce.

**Admin:** PhoenixWP → Gift Product
